# Machine Operation

### Controlling machines with Alexa
Most things in the arcade can be controlled by issuing commands to Alexa. “__Alexa, turn on \_\_\_\___.” or “__Alexa, turn off \_\_\_\___.”

“**Everything**”: Turns on/off all machines, as well as fans.

#### Machines by Name
- **Main ITG**: ITG in the main room. This will power on the machine. You can then turn on/off the computer with the button above the monitor.
- **Side ITG**: ITG in the side room. This will power on the machine. You can then turn on/off the computer with the button above the monitor.
